# Task Manager (Role-Based Access)

## Features
- Admin: 
    - Create users
    - Assign tasks to users
    - Set task priority
    - View users list
- User:
    - View assigned tasks
    - Update task status
- JWT Authentication
- Role-based routing

## Tech Stack
- MongoDB
- Express.js
- React (Vite)
- Node.js
- JWT
